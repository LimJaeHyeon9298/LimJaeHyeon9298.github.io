2.Swift의 데이터 타입

1) INT : 정수(Integer) ★                            // -1,0.1
2) Float : 실수(부동소수점) -> 6자리 소수                           
3) Double : 실수(부동소수점) -> 15자리 소수점★           //3.141592
4) Character : 문자(글자 한개)                       //"A","한", " " (빈문자열)
5) String : 문자열★
6) Bool : 참과 거짓★                               // true(참), false(거짓)

7) 기타 : UInt, UInt32, UInt16, UInt8: 0, 그리고 양의정수




var k : Int = 3

type(of:k)  // 타입을 확인하는 방법
